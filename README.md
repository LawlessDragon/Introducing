## 🚀 Getting Started

To view this website locally:

1. Clone the repository
```bash
git clone https://github.com/LawlessDragon/Introducing.git
```

2. Navigate to the project folder
```bash
cd Introducing
```

3. Open `index.html` in your preferred browser
